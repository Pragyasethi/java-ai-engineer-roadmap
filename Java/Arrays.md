#Arrays

An array is a fixed-size collection of elements of the same data type stored in contiguous memory locations.

Imagine a train with 5 coaches.

Each coach has a number:

* Coach 0
* Coach 1
* Coach 2
* Coach 3
* Coach 4

If you want the passenger in Coach 3, you directly go to Coach 3.

An array works the same way using indexes.

Why do we need it?

To store multiple values of the same type and access them quickly using an index.

Important Points

* Fixed size
* Same data type
* Index starts at 0
* Fast access using index (O(1))

Real-world Usage

* Student marks
* Monthly sales
* Sensor readings

Memory Trick

Array = Train.
Index = Coach Number.
