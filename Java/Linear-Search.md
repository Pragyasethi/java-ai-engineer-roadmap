#Linear Search

Definition

Linear Search checks every element one byone until the required element is found.

Imagine searching for your friend in a queue.

You ask every person until you find them.

Why do we need it?

When data is not sorted.

Important Points

* Works on unsorted data
* Simple to implement
* Time Complexity: O(n)

Real-world Examples

Searching a name in an attendance sheet.
